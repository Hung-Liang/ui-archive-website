# 🎉 しぐれういアーカイブサイト

- しぐれういのYouTubeライブ配信のアーカイブを振り返ることに特化したウェブサイトです。
- YouTube APIを使用して毎日自動的に更新されます。

## 🌐 言語

* [English](docs/README.en.md)
* [繁體中文](docs/README.zh-TW.md)
* [日本語](docs/README.jp.md)

## 🚀 利用方法

- しぐれういアーカイブサイトを閲覧してください。 [しぐれういアーカイブサイト](https://ui-archive-website.bobing.dev/index.html)。
- サイトは最新の配信アーカイブリストを自動で読み込みます。
- 見たい年と月を選択し、該当する配信アーカイブをクリックしてください。

## 📱 PWA (Progressive Web App) 対応

当サイトは PWA に対応しています。スマートフォンやデスクトップに「インストール」することで、通常のアプリのように素早く起動できます。

### 💡 インストール方法

- **iOS (Safari)**: 画面下部の「共有」ボタンをタップし、「ホーム画面に追加」を選択します。
- **Android (Chrome)**: 右上の 3 つの点（メニュー）をタップし、「アプリをインストール」または「ホーム画面に追加」を選択します。
- **PC (Chrome/Edge)**: アドレスバー右側の「インストール」アイコンをクリック（またはメニューからインストールを選択）します。
