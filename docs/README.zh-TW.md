# 🎉 しぐれうい回顧網站

- 這是一個網站，專門用來回顧しぐれうい的 YouTube 直播回放。
- 網站會自動每天透過YouTube API更新上傳。

## 🌐 語言

* [English](docs/README.en.md)
* [繁體中文](docs/README.zh-TW.md)
* [日本語](docs/README.jp.md)

## 🚀 使用方法

- 瀏覽 [しぐれうい回顧網站](https://ui-archive-website.bobing.dev/index.html)。
- 網站會自動載入最新的直播回放列表。
- 選擇你要看的年份跟月份，然後點擊相應的直播回放。

## 📱 PWA (Progressive Web App) 支援

本網站已支援 PWA，您可以將其「安裝」到手機或桌面上，像一般 App 一樣快速啟動。

### 💡 如何安裝？

- **iOS (Safari)**: 點擊瀏覽器底部的「分享」按鈕，選擇「加入主畫面」。
- **Android (Chrome)**: 點擊右上角的三個點，選擇「安裝應用程式」或「加入主畫面」。
- **電腦版 (Chrome/Edge)**: 點擊網址列右側的「安裝」圖示（或在選單中選擇安裝）。
